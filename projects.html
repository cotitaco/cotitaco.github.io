Analyzing data streams to prevent DNS attacks

The purpose of this project was to simulate a real flow of constant data and dynamically train a ML or DL model based on this data. When I say dynamically I mean to use time windows and evaluate every x time the performance of our model. If the performace metric goes below a threshold, then we will re-train the model. For this, I created a docker instance which already had kafka server. Then I used the API in python to read the incoming data, train my model and keep analyzing the performance metrics to decide whether it is necessary to retrain the model and keep on working on an "Online-learning" way or if it is better to do batch- learning. For more details please contact me directly.

Data Augmentation via sketches comparing cGANs and CycleGANs

This project is a derivation of a problem when training Generative Adversarial Networks to generte images with conditioning inputs. In this example I am training a cGAN (conditional GAN) to transform a monocromatic sketch into an Impressionist-like painting, as if Van Gogh or Monet has painted it. Now what is the problem I found. During that time period, Impressionist artist did not painted castles. So obviously when I input a sketch of a landscape with a castle, the GAN will output something like a mount or a mountine. To handle this problem I then compared the efficiency of a conditional GAN and a CycleGAN to generate Impressionist castles based on a few examples I could find online. To train my models, I downloaded only 50 Impressionist-like castles, and then I transformed these images into sketches to have the condition. Then after only training them with 50 pairs of images, I asked the models to generate thousands of castle examples. For more information in this type of Data Augmentation please contact me.

Improving Lottery Ticket Hypothesis with Knowledge Distillation

In 2018, Dr Hinton together with a group of researchers published a intriging paper which described a prunning technique for dense neural networks. This technique called Lottery Ticket Hypothesis consisted of reducing the links between some neurons my creating masks and finding an optimal sparse sub network which can perform predictions as good as the original and in some cases even better. In our research my team and I combined this technique with knowledge distillation which consists of having a dense teacher model, which will train a studend model which is smaller ( may have a different and smaller architecture or could be a trimmed version of the same arch). And now train the student with respect to the the labels that the teacher gives. To learn more about our process and results please contact me.
