Industry Projects' Description

Data & Analytics

Project 1

The purpose of this project was to process batch information from the client stored in a HUE database and, using a pre-trained Gradient Boosting model, determine whether that specific client should have a payroll loan. Technologies used are Python, Cloudera, HUE as the database, Spark, and Autosys as the scheduler. We also used Agile methodology, reporting all steps in JIRA and storing the code in a private GIT.

My role in this project was first a developer, translating code from SAS to Python and Spark, but then I became tech lead; I was in charge of administrating together with the PM the activities of the developers and having meetings with other areas to ensure the correct deployment of the product to production servers.

Project 2

This project was my first baby. It started with a POC and me acting as a Business Analyst, understanding its needs. The project involved analyzing Excel files and passing them to an HDFS platform in Hadoop.
